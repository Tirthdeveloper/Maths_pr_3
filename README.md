# 📘 Maths Project 3

A Python-based Mathematics & Statistics project containing practical implementations of important statistical concepts, probability calculations, and data analysis techniques using Python libraries.

---

## 🚀 Features

- 📊 Descriptive Statistics
- 📈 Covariance & Correlation
- 📉 ANOVA Test
- 🎲 Probability Calculations
- 🧠 Bayes Theorem
- 📋 Data Cleaning & Processing
- 🐍 Python Implementation with NumPy & Pandas

---

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- SciPy
- Jupyter Notebook

## ▶️ How to Run

### Clone Repository

```bash
git clone https://github.com/Tirthdeveloper/Maths_pr_3.git
```

### Move into Project Folder

```bash
cd Maths_pr_3
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python main.py
```
